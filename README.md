LIFE CARE – Patient & Clinic Management Application

A modern web application designed to help hospitals manage patient registration, login, patient records, and appointment booking.  
The project provides a clean, responsive, and user-friendly experience.

 FEATURES
 
PATIENT FEATURES 

- Create account (Sign Up)
- Login system with validation
- Toast notifications for feedback
- Manage profile details
- Book appointments

Clinic Features
- Appointment booking interface
- About Us / Biography page
- Smooth navigation
- Responsive design
- 

TOOLS AND TECHNOLOGIES USED

 - React.js
 - Vite
 - React Router (navigation & routing)
 - React Toastify
 - JavaScript (ES6)
 - CSS
 - LocalStorage (temporary database)
 - React.js – core library
  -VS Code – development environment
